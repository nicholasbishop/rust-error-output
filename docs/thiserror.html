This is a reference to help answer this question: ‚Äúwhat will my Rust program print if something goes wrong?‚Äù

The examples were compiled with Rust 1.46.0. All output in these examples goes to stderr, nothing goes to stdout.

Setup code for thiserror

#[derive(Debug, thiserror::Error)]
enum CustomError {
    #[error("bad thing: {0}")]
    BadThing(#[from] std::io::Error),
}

fn make_error() -> Result<(), CustomError> {
    Ok(std::fs::remove_file("/this/file/does/not/exist")?)
}

Debug

fn main() {
    eprintln!("{:?}", make_error().unwrap_err())
}

BadThing(Os { code: 2, kind: NotFound, message: "No such file or directory" })

Display

fn main() {
    eprintln!("{}", make_error().unwrap_err())
}

bad thing: No such file or directory (os error 2)
