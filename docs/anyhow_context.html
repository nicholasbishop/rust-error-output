This is a reference to help answer this question: “what will my Rust program print if something goes wrong?”

The examples were compiled with Rust 1.46.0. All output in these examples goes to stderr, nothing goes to stdout.

Setup code for anyhow with context

use anyhow::Context;

fn make_error() -> Result<(), anyhow::Error> {
    Ok(std::fs::remove_file("/this/file/does/not/exist").context("oh no")?)
}

Debug

fn main() {
    eprintln!("{:?}", make_error().unwrap_err())
}

oh no

fn main() {

}
