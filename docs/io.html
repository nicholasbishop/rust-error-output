This is a reference to help answer this question: ‚Äúwhat will my Rust program print if something goes wrong?‚Äù

The examples were compiled with Rust 1.46.0. All output in these examples goes to stderr, nothing goes to stdout.

Setup code for std::io::Error

fn make_error() -> Result<(), std::io::Error> {
    std::fs::remove_file("/this/file/does/not/exist")
}

Debug

fn main() {
    eprintln!("{:?}", make_error().unwrap_err())
}

Os { code: 2, kind: NotFound, message: "No such file or directory" }

Display

fn main() {
    eprintln!("{}", make_error().unwrap_err())
}

No such file or directory (os error 2)

Unwrap

fn main() {
    make_error().unwrap();
}
