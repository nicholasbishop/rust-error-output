This is a reference to help answer this question: ‚Äúwhat will my Rust program print if something goes wrong?‚Äù

The examples were compiled with Rust 1.46.0. All output in these examples goes to stderr, nothing goes to stdout.

Setup code for stable_eyre

fn make_error() -> Result<(), stable_eyre::eyre::Report> {
    Ok(std::fs::remove_file("/this/file/does/not/exist")?)
}

fn install_hook() -> stable_eyre::eyre::Result<()> {
    let hook =
        stable_eyre::HookBuilder::default().capture_backtrace_by_default(true);
    hook.install()
}
